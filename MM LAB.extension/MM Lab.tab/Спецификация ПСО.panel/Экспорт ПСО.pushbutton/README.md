# Экспорт ПСО

Экспорт спецификации помещений (ПСО) из Revit в Excel с группировкой и объединением ячеек по квартирам.

## Возможности

- Сбор всех размещённых помещений (Rooms) из текущего документа Revit
- Группировка помещений по квартирам (корпус, номер квартиры, этаж, секция)
- Экспорт в формат `.xlsx` с форматированием и объединением ячеек
- Автоматическое открытие файла после экспорта

## Используемые параметры

### Уровень квартиры (объединяются по вертикали)

| Параметр | Описание |
|---|---|
| `GP_23_НомерКорпуса` | Номер корпуса |
| `GP_23_НомерКв` | Номер квартиры |
| `GP_23_Назначение` | Назначение |
| `GP_01_Этаж_Номер` | Этаж расположения |
| `GP_23_НомерСекции` | Номер секции (подъезда) |
| `GP_23_ПлКвОбщая+НеотБезКоэф_ПСО` | Общая площадь квартиры без понижающего коэффициента |
| `GP_23_ПлКвОбщая+Неот_ПСО` | Общая площадь квартиры с понижающим коэффициентом |
| `GP_23_КолвоКомнат` | Количество комнат |
| `GP_23_ПлКвЖилая_ПСО` | Жилая площадь квартиры |
| `Полная высота` | Высота потолков |

### Уровень помещения (отдельная строка)

| Параметр | Описание |
|---|---|
| `GP_23_НомерПомКв` | Номер помещения в квартире |
| `Имя` | Наименование помещения |
| `GP_23_Площадь_ПСО` | Площадь помещения без коэффициента |
| `GP_23_ПлощадьСКоэф_ПСО` | Площадь помещения с коэффициентом |
